## 📌 Project Overview

This project explores student academic performance using **Python (Jupyter Notebook)** and **Tableau**.
The analysis focuses on identifying patterns, relationships, and factors that influence student success.




## 🎯 Objectives

* Analyze student data to uncover performance trends
* Clean and preprocess raw data
* Visualize distributions and relationships
* Build an interactive dashboard
* Provide actionable insights for decision-making

---

## 🧰 Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* Jupyter Notebook
* Tableau Public

## 💡 Key Insights

* Students with fewer absences perform better
* Higher parental education correlates with higher grades
* Internet access slightly improves academic performance
* High-risk students tend to score lower
